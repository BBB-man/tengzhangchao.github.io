RSA加密算法解析

发表于 2016-09-06   |   分类于 技术交流   |  

RSA加密算法属于公钥加密算法，是一种非对称密码算法，所谓非对称，就是一个密码用来加密，另一个密码用来解密，一般来说，用公钥加密，私钥解密。

算法原理

RSA算法基于一个事实：将两个大素数相乘十分容易，但是想要对其乘积进行因式分解却极其困难，因此可以将乘积公开作为加密密钥。

RSA算法三个参数：n、e1、e2

公式算法介绍：

n=p*q （p、q为2个大质数） n的二进制所占用的位数即秘钥的长度。

e1与e2是一对相关的值，e1可以任意取，但要求e1与(p-1)(q-1)互质；要求(e2e1)mod((p-1)*(q-1))=1.

(n,e1),(n,e2)就是密钥对，其中(n,e1)为公钥，(n,e2)为私钥。

RSA算法加解密完全相同：

A:明文
B:密文

——用公钥加密——
A=B^e2 mod n #解密
B=A^e1 mod n #加密

——用私钥加密——
A=B^e1 mod n
B=A^e2 mod n
